# 单题评分页面优化记录

文件：`dist/index.html`

---

## 第一轮优化

### 1. Ctrl+Enter 快捷评分
- 按钮文字后显示 `(Ctrl+Enter)` 提示
- 全局键盘监听，仅单题评分页且非评分中时生效
- 代码：`document.addEventListener('keydown', ...)` + 按钮内 `<span>`

### 2. 空状态引导
- 结果区域隐藏时，左栏自动展开到 100% 宽度
- 移除了不可达的空状态代码（`v-if="!grading && !gradeResult"` 被外层条件覆盖）
- 通过按钮上的快捷键提示替代静态引导文字

### 3. Loading 状态
- 选择题目时 `loadingQuestions = true`，完成后重置
- 评分按钮已有 `:loading="grading"` 状态
- AI 生成答案按钮已有 `:loading="genAnswerLoading"` 状态

### 4. 移除空的 el-menu
- 侧边栏曾有一个空的 `<el-menu>` 标签，无实际内容，已删除

### 5. 结果区域可折叠
- 评分结果头部增加折叠/展开按钮
- `resultCollapsed` 状态控制 `v-show`
- 折叠后左栏自动扩展到 100% 宽度（通过动态 width 计算）

---

## 第二轮优化

### 6. 题目选择器 loading 指示
- 给侧边栏题目 `<el-select>` 加了 `:loading="loadingQuestions"`
- 题目列表加载中时下拉框显示 loading 转圈

---

## 第三轮优化

### 7. 评分结果区高度压缩
- **第一版**：分数从 56px 居中大字改为 36px 左对齐 + 横向进度条
- **第二版**（最终）：分数、进度条、置信度合并为一行紧凑排列
  - 分数 28px 粗体 + "/满分 分"
  - 进度条 `stroke-width=6`，`max-width=160px`，隐藏百分比文字
  - 置信度改为 `size="small"` 标签，只显示百分比数字
  - padding 从 20px 减至 8px，margin-bottom 从 16px 减至 8px
  - 整体高度从约 120px 压缩到约 40px
