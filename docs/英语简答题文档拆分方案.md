# 英语简答题文档拆分方案

> 用途：将英语简答题参考标准 Word 文档按题目拆分为独立文件，并提取通用评分规则

## 概述

原始文档 `（英语提交）4.2普测简答题参考标准.docx` 包含 12 道试题，每题结构如下：

```
模块标题（如：英语基础模块M3 U1【...】）
【000x】题目名称
阅读材料（多个段落）
简答题问题（答案均出自原文）
第1题 / 第2题 / 第3题（问题 + 参考答案）
第一步：逐项判定（各题评分细则）      ← 各题不同
第二步：计分规则（通用）              ← 所有题相同
第三步：输出格式（通用）              ← 所有题相同
```

## 拆分规则

### 1. 按题目拆分（12 个文件）

- 每题从 **模块标题** 开始，到 **第一步逐项判定** 结束（不含第二步、第三步）
- 切割边界：`模块标题` 段落 → `第三步` 之前的段落（不含"第二步"及之后内容）
- 输出命名：`（英语提交）4.2普测简答题参考标准-第N题.docx/.pdf`

### 2. 通用评分规则（1 个文件）

- 从 **第二步：计分规则** 到 **第三步：输出格式**
- 12 道题这部分完全一样，只保留一份
- 输出命名：`（英语提交）4.2普测简答题参考标准-通用第2-3步.docx/.pdf`

## 拆分脚本

```python
"""
英语简答题文档拆分脚本
用法：将原始文档放在同目录下，直接运行
"""
from docx import Document
from docx.oxml.ns import qn
from reportlab.lib.pagesizes import A4
from reportlab.pdfbase import pdfmetrics
from reportlab.pdfbase.ttfonts import TTFont
from reportlab.lib.styles import getSampleStyleSheet, ParagraphStyle
from reportlab.platypus import SimpleDocTemplate, Paragraph, Spacer
from reportlab.lib.units import mm
from reportlab.lib.enums import TA_CENTER
from xml.sax.saxutils import escape
import os
import re

# ========== 配置 ==========
SRC_FILE = '（英语提交）4.2普测简答题参考标准.docx'
FONT_PATH = '/System/Library/Fonts/STHeiti Light.ttc'  # macOS 中文字体
OUTPUT_DIR = '.'  # 输出目录

# ========== 注册中文字体 ==========
pdfmetrics.registerFont(TTFont('ChineseFont', FONT_PATH, subfontIndex=0))

# ========== PDF 样式 ==========
STYLES = getSampleStyleSheet()
TITLE_STYLE = ParagraphStyle('T', parent=STYLES['Normal'],
    fontName='ChineseFont', fontSize=14, leading=20,
    alignment=TA_CENTER, spaceAfter=6*mm)
HEADING_STYLE = ParagraphStyle('H', parent=STYLES['Normal'],
    fontName='ChineseFont', fontSize=12, leading=17,
    spaceBefore=4*mm, spaceAfter=2*mm)
BODY_STYLE = ParagraphStyle('B', parent=STYLES['Normal'],
    fontName='ChineseFont', fontSize=10, leading=15,
    spaceAfter=1*mm)


def copy_para(src_para, dst_para):
    """精确复制段落格式和 run 样式（字体、字号、加粗、斜体、下划线、颜色）"""
    dst_para.alignment = src_para.alignment
    pf = src_para.paragraph_format
    if pf.space_before is not None:
        dst_para.paragraph_format.space_before = pf.space_before
    if pf.space_after is not None:
        dst_para.paragraph_format.space_after = pf.space_after
    if pf.line_spacing is not None:
        dst_para.paragraph_format.line_spacing = pf.line_spacing

    for run in src_para.runs:
        nr = dst_para.add_run(run.text)
        nr.bold = run.bold
        nr.italic = run.italic
        nr.underline = run.underline
        if run.font.size:
            nr.font.size = run.font.size
        if run.font.name:
            nr.font.name = run.font.name
        if run.font.color and run.font.color.rgb:
            nr.font.color.rgb = run.font.color.rgb

    if not src_para.runs:
        dst_para.add_run(src_para.text)


def gen_pdf(doc, pdf_name):
    """从 docx Document 对象生成 PDF"""
    story = []
    for para in doc.paragraphs:
        text = para.text.strip()
        if not text:
            story.append(Spacer(1, 2*mm))
            continue
        escaped = escape(text)

        # 检测标题/加粗段落
        if para.alignment == 1:  # 居中
            story.append(Paragraph(escaped, TITLE_STYLE))
        elif text.startswith('第') and ('题' in text[:5] or '步' in text[:5]):
            story.append(Paragraph(f'<b>{escaped}</b>', HEADING_STYLE))
        elif '【' in text and '】' in text and len(text) < 30:
            story.append(Paragraph(f'<b>{escaped}</b>', HEADING_STYLE))
        elif any(text.startswith(p) for p in ['英语基础模块', '简答题问题', '请严格按照']):
            story.append(Paragraph(f'<b>{escaped}</b>', HEADING_STYLE))
        else:
            parts = []
            for run in para.runs:
                t = escape(run.text)
                parts.append(f'<b>{t}</b>' if run.bold else t)
            combined = ''.join(parts) if parts else escaped
            story.append(Paragraph(combined, BODY_STYLE))

    SimpleDocTemplate(pdf_name, pagesize=A4,
        leftMargin=20*mm, rightMargin=20*mm,
        topMargin=20*mm, bottomMargin=20*mm).build(story)


def split_questions(src_file, output_dir='.'):
    """主函数：拆分文档"""
    src = Document(src_file)
    total = len(src.paragraphs)

    # 1. 自动检测各题的【000x】标记位置
    markers = []
    for i, p in enumerate(src.paragraphs):
        text = p.text.strip()
        m = re.search(r'【(\d{4})】', text)
        if m:
            markers.append((i, m.group(1)))

    print(f'原始文档共 {total} 段，检测到 {len(markers)} 道题目')

    # 2. 构建每题的切割范围（从模块标题到"第二步"之前）
    question_ranges = []
    for idx, (marker_pos, qid) in enumerate(markers):
        start = marker_pos - 1  # 模块标题在标记前一段

        # 找到"第二步：计分规则"作为截止点
        cutoff = total
        for j in range(marker_pos, total):
            if '第二步：计分规则' in src.paragraphs[j].text.strip():
                cutoff = j
                break

        question_ranges.append((start, cutoff, qid))

    # 3. 找到通用的第二步+第三步范围
    step2_start = None
    step3_end = None
    for i, p in enumerate(src.paragraphs):
        text = p.text.strip()
        if step2_start is None and '第二步：计分规则' in text:
            step2_start = i
        if step2_start is not None and '第三步：输出格式' in text:
            # 找到"简要说明..."那一段
            for j in range(i + 1, min(i + 5, total)):
                if '简要说明' in src.paragraphs[j].text.strip():
                    step3_end = j + 1
                    break
            if step3_end is None:
                step3_end = i + 2
            break

    # 4. 生成各题文件
    for qi, (start, end, qid) in enumerate(question_ranges):
        qnum = qi + 1
        dst = Document()
        for i in range(start, end):
            copy_para(src.paragraphs[i], dst.add_paragraph())

        docx_name = os.path.join(output_dir,
            f'（英语提交）4.2普测简答题参考标准-第{qnum}题.docx')
        dst.save(docx_name)

        pdf_name = docx_name.replace('.docx', '.pdf')
        gen_pdf(dst, pdf_name)

        print(f'Q{qnum:2d} [{qid}]: 段落 {start}-{end-1} '
              f'→ {os.path.getsize(docx_name)//1024}KB + '
              f'{os.path.getsize(pdf_name)//1024}KB')

    # 5. 生成通用评分规则文件
    if step2_start and step3_end:
        dst = Document()
        for i in range(step2_start, step3_end):
            copy_para(src.paragraphs[i], dst.add_paragraph())

        docx_name = os.path.join(output_dir,
            '（英语提交）4.2普测简答题参考标准-通用第2-3步.docx')
        dst.save(docx_name)

        pdf_name = docx_name.replace('.docx', '.pdf')
        gen_pdf(dst, pdf_name)

        print(f'通用2-3步: 段落 {step2_start}-{step3_end-1} '
              f'→ {os.path.getsize(docx_name)//1024}KB + '
              f'{os.path.getsize(pdf_name)//1024}KB')

    print(f'\n完成！共生成 {len(question_ranges) + 1} 个文件（Word + PDF）')


if __name__ == '__main__':
    split_questions(SRC_FILE, OUTPUT_DIR)
```

## 结构正则模板（通用识别工具）

以下正则可用于解析任意英语简答题文档的结构，不依赖具体段落编号：

```python
import re

# ===== 一级结构正则（文档骨架） =====
PATTERNS = {
    # 模块标题行
    'module_title': r'^英语基础模块\s*M\d+\s+U\d+【高等教育出版社】【教材发展研究所】【语篇知识/文化素养】【难】$',

    # 题目编号标记（捕获4位数字ID + 可选标题）
    'question_id': r'^【(\d{4})】\s*(.*)$',

    # 问题区标题
    'questions_header': r'^简答题问题（答案均出自原文）$',

    # 评分指令
    'scoring_instruction': r'^请严格按照以下规则对学生答案进行评分：$',

    # 三步标题
    'step1': r'^第一步：逐项判定$',
    'step2': r'^第二步：计分规则$',
    'step3': r'^第三步：输出格式$',
}

# ===== 二级结构正则（评分细则内部） =====
SCORING_PATTERNS = {
    # 子题标题：第1题 / 第2题 / 第3题（可能与"问题："合并）
    'sub_question': r'^第[123]题\s*(问题：.*)?$',

    # 问题行
    'question_line': r'^问题：.+$',

    # 参考答案
    'reference_answer': r'^参考答案：.+$',

    # 采分点说明
    'scoring_points': r'^采分点（关键词）：.+$',

    # 得分表头（可能有tab也可能没有）
    'score_table_header': r'^得分\t?条件$',

    # 得分行
    'score_row': r'^([012])分\t(.+)$',

    # 同义表达（可选段落）
    'synonyms': r'^同义表达可接受范围：?$',

    # 示例
    'examples': r'^示例：?\s*$',

    # 子采分点 (A)(B)(C)...
    'sub_point': r'^\(([A-Z])\)\s*(.+)$',

    # 注意事项（可选段落）
    'note': r'^注意：(.+)$',
}

# ===== 用法示例 =====
def parse_doc_structure(docx_path):
    """解析文档结构，返回段落分类"""
    from docx import Document
    doc = Document(docx_path)
    structure = []
    for i, para in enumerate(doc.paragraphs):
        text = para.text.strip()
        if not text:
            structure.append((i, 'empty', text))
            continue
        matched = False
        for name, pattern in {**PATTERNS, **SCORING_PATTERNS}.items():
            if re.match(pattern, text):
                structure.append((i, name, text[:60]))
                matched = True
                break
        if not matched:
            structure.append((i, 'content', text[:60]))
    return structure
```

## Word 格式保留方案

### 问题

纯文本存储无法保留格式。从 Word 导入后，加粗、字号、颜色全部丢失。
导出时用正则只能识别结构，无法恢复原始格式。

### 方案：Word → HTML → 系统存储 → HTML → Word

**导入流程**：
```
Word 文件 → python-docx 解析 → 提取 HTML（保留格式）→ 存入 content_html 字段
                              → 提取纯文本 → 存入 content 字段（评分引擎用）
```

**导出流程**：
```
content_html → html2docx / pandoc → Word 文件（格式还原）
```

**核心函数**（python-docx → HTML 转换）：

```python
from docx import Document

def docx_to_html(docx_path):
    """将 Word 文档转换为保留格式的 HTML"""
    doc = Document(docx_path)
    html_parts = []

    for para in doc.paragraphs:
        if not para.text.strip():
            html_parts.append('<p><br></p>')
            continue

        runs_html = []
        for run in para.runs:
            text = run.text.replace('&', '&amp;').replace('<', '&lt;').replace('>', '&gt;')
            styles = []

            if run.bold:
                text = f'<strong>{text}</strong>'
            if run.italic:
                text = f'<em>{text}</em>'
            if run.underline:
                text = f'<u>{text}</u>'
            if run.font.size:
                pt = run.font.size.pt
                styles.append(f'font-size:{pt}pt')
            if run.font.color and run.font.color.rgb:
                hex_color = str(run.font.color.rgb)
                styles.append(f'color:#{hex_color}')
            if run.font.name:
                styles.append(f'font-family:{run.font.name}')

            if styles:
                text = f'<span style="{";".join(styles)}">{text}</span>'
            runs_html.append(text)

        # 段落对齐
        align_map = {1: 'center', 2: 'right', 3: 'justify'}
        align = align_map.get(para.alignment, '')
        align_attr = f' style="text-align:{align}"' if align else ''

        html_parts.append(f'<p{align_attr}>{"".join(runs_html)}</p>')

    return '\n'.join(html_parts)


def html_to_docx(html_content, output_path):
    """将 HTML 内容转换回 Word 文档"""
    # 方案1：使用 pandoc（最准确）
    # subprocess.run(['pandoc', '-f', 'html', '-t', 'docx', '-o', output_path], input=html_content)

    # 方案2：使用 python-docx 简化版（不支持复杂HTML）
    from docx import Document
    import re
    doc = Document()

    # 按 <p> 分段
    paragraphs = re.findall(r'<p[^>]*>(.*?)</p>', html_content, re.DOTALL)
    for p_html in paragraphs:
        para = doc.add_paragraph()
        # 简单提取文本和粗体
        parts = re.split(r'(</?(?:strong|b|em|i|u)[^>]*>)', p_html)
        bold = False
        italic = False
        underline = False
        for part in parts:
            if part in ('<strong>', '<b>'): bold = True
            elif part in ('</strong>', '</b>'): bold = False
            elif part in ('<em>', '<i>'): italic = True
            elif part in ('</em>', '</i>'): italic = False
            elif part == '<u>': underline = True
            elif part == '</u>': underline = False
            elif not part.startswith('<'):
                clean = re.sub(r'<[^>]+>', '', part)
                if clean:
                    run = para.add_run(clean)
                    run.bold = bold
                    run.italic = italic
                    run.underline = underline

    doc.save(output_path)
```

## Word 导入功能

### 功能概述

支持将拆分后的单题 `.docx` 文件或原始完整 12 题 `.docx` 文档直接导入题库系统。
解析基于正则模板自动识别文档结构，无需手动配置段落编号。

### API

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/import-word` | 上传 .docx 文件，返回预览 + token |
| POST | `/api/import-word/confirm` | 通过 token 确认导入，写入数据库 |

### 字段对应关系

Word 解析器 (`app/word_parser.py`) 输出 → 数据库 `questions` 表字段：

| Word 解析字段 | 数据库字段 | 格式 | 说明 |
|---|---|---|---|
| `question_number` | `question_number` | 纯文本 | 题号，如 `0001` |
| `title` | `title` | 纯文本 | 标题，如 `Chinese New Year` |
| `content` | `content` | 纯文本 | 阅读材料 + 问题 + 参考答案（不含评分细则），供评分引擎使用 |
| `content` | `original_text` | — | （**已废弃**，早期版本写入相同值，现已改为 HTML） |
| `content_html` | `content_html` | HTML | 阅读材料 + 问题 + 参考答案（保留格式：粗体/字号/颜色/字体），供前端展示 |
| `original_text` | `original_text` | HTML | 完整原始文档（含模块标题 + 题号 + 阅读材料 + 问题 + 答案 + 评分细则），保留格式 |
| `standard_answer` | `standard_answer` | 纯文本 | 从评分细则中 `参考答案：` 行自动提取 |
| `rubric_script` | `rubric_script` | 纯文本 | 第一步逐项判定（评分细则） |
| `rubric_script` | `rubric_rules` | 纯文本 | 同上，两个字段写入相同值 |
| `rubric_points` | `rubric_points` | 纯文本 | 单题文件中为空字符串 |
| `max_score` | `max_score` | 数值 | 固定 6（3 小题 × 2 分） |
| `subject` | `subject` | 纯文本 | 固定 `english` |
| `exam_name` | `exam_name` | 纯文本 | 模块标题，如 `英语基础模块M3 U1【高等教育出版社】【...】` |
| `difficulty` | `difficulty` | 纯文本 | 从模块标题末尾【难/中/易】提取，映射为 hard/medium/easy |
| — | `rubric` | JSON | 手动构造 `{"contentType":"简答题","knowledge":"","standardAnswer":"..."}` |

### 三个文本字段对比

以第 1 题 (Chinese New Year) 为例：

| 字段 | 格式 | 内容范围 | 字符数 |
|---|---|---|---|
| `content` | 纯文本 | 阅读材料 + 问题 + 答案（不含评分细则） | 1,527 |
| `content_html` | HTML | 同上，保留格式 | 4,009 |
| `original_text` | HTML | 全部（含模块标题 + 题号 + 阅读材料 + 问题 + 答案 + 第一步评分细则） | 12,543 |

### 各字段用途

- **`content`**（纯文本）— 评分引擎三层并行（关键词/向量/LLM）使用，不能有 HTML 标签
- **`content_html`**（HTML）— 前端题库列表、查看对话框展示，用 DOMPurify 清洗后渲染
- **`original_text`**（HTML）— 完整原始文档，用于题目回溯、格式校验、后续导出

### HTML 格式保留内容

通过 `_docx_to_html_by_paragraphs()` 函数提取，保留以下格式：

- `<strong>` — 加粗
