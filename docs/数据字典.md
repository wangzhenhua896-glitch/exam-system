## questions（题目）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK | 题目 ID |
| subject | TEXT | NOT NULL | 科目：politics / chinese / english / general |
| title | TEXT | NOT NULL | 题目标题（取 content 前30字） |
| content | TEXT | NOT NULL | 题干内容 |
| rubric | TEXT | NOT NULL | 评分标准 JSON（含 rubricRules / points / rubricScript 等） |
| max_score | REAL | DEFAULT 10.0 | 满分值 |
| standard_answer | TEXT | | 标准答案 |
| original_text | TEXT | | 原始未处理原文；英语题 Word 导入后存完整原始文档 HTML（含模块标题+阅读材料+评分细则），其他科目存粘贴的纯文本 |
| rubric_rules | TEXT | | 评分规则文字描述 |
| rubric_points | TEXT | | 分数分布（每行一个要点+分值，如"尊老爱幼 (1.6分)"） |
| rubric_script | TEXT | | 结构化评分脚本（评分引擎的唯一依据） |
| quality_score | REAL | | AI 评估的试题科学性分数 |
| question_number | TEXT | | 题号（如"1"、"2"） |
| difficulty | TEXT | | 难度：easy / medium / hard |
| exam_name | TEXT | | 考试名称 |
| parent_id | INTEGER | FK → questions(id) | 父题 ID（NULL=独立题/父题，非NULL=子题） |
| scoring_strategy | TEXT | | 单题评分策略覆盖：max/min/avg/median（NULL 使用全局默认；英语子题固定 max） |
| content_html | TEXT | | HTML格式题干（保留Word导入的粗体/字号/颜色等格式），前端展示用；content 字段为纯文本供评分引擎使用 |
| question_type | TEXT | DEFAULT 'essay' | 题型标识：essay/single_choice/multi_choice/fill_blank/true_false/translation |
| workflow_status | TEXT | | 英语编辑器5步工作流状态 JSON（独立列，不存 rubric 中避免覆盖） |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 更新时间 |

## question_answers（满分答案库）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PK | 记录 ID |
| question_id | INTEGER | NOT NULL, FK → questions(id) | 关联题目 |
| scope_type | TEXT | NOT NULL, DEFAULT 'question' | 粒度：question（整题级）/ scoring_point（采分点级） |
| scope_id | TEXT | DEFAULT '' | 采分点编号（如 "1"、"2"；英语子题用 "Q1"、"Q2"、"Q3"） |
| score_ratio | REAL | DEFAULT 1.0 | 占该采分点满分的百分比（1.0=满分，0.5=半分） |
| answer_text | TEXT | NOT NULL | 答案文本（用于匹配度计算）；英语 scope_type=scoring_point 时存采分点 JSON |
| label | TEXT | DEFAULT '' | 标注（如"标准答案"、"等价表述1"、"采分点A"） |
| source | TEXT | DEFAULT 'manual' | 来源：manual / ai_generated / student_real / migrated |
| sort_order | INTEGER | DEFAULT 0 | 排序 |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

## grading_params（评分参数配置）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| key | TEXT | PK | 参数名 |
| value | TEXT | NOT NULL | 参数值 |
| description | TEXT | DEFAULT '' | 说明 |
| updated_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 更新时间 |

默认参数：

| key | 默认值 | 说明 |
|-----|--------|------|
| match_pass_threshold | 0.80 | 匹配度≥此值直接给分 |
| match_review_threshold | 0.65 | 匹配度在此区间→LLM复核 |
| full_check_threshold | 0.72 | 整题评分校验阈值 |
| scoring_strategy | avg | 全局评分策略：max/min/avg/median |
